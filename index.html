Variable de entorno en Vercel

Cómo activar en producción (4 pasos) 1. Crea cuenta en payphone.app/para-negocios → obtén tu Store ID y Token.
2. Crea cuenta en resend.com → verifica tu dominio → obtén API Key.
3. En Vercel → Settings → Environment Variables pega las 4 keys.
4. Sube el archivo api/confirm-order.js (incluido como comentario al final del HTML).
Con eso el flujo completo funciona: pago → webhook → correo automático.
